# Vira — Marketing Site

The first app built for how teens with ADHD actually think. Marketing landing for the Vira brand & app (TFG project, ESDAPC 2024/25, with Fundació Adana).

Built with **Vite + React + TypeScript + Three.js (R3F) + GSAP + Lenis**.

## Dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
npm run preview
```

## Project

- `src/three/` — R3F scenes (hero compass, mascot morph, feature tiles, orb field)
- `src/sections/` — page sections (Hero, Problem, Features, Mascot, HowItWorks, Backed, CTA, Footer)
- `src/components/` — buttons, logo pill
- `src/styles/` — tokens + globals
- `public/viri/` — 3D mascot frames
- `public/logo.svg` — Vira wordmark
